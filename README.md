# Korea Stock News Research Assistant

한국 주식 뉴스 기사를 수집하고, 관련 종목 및 ETF 후보를 매칭하며, AI 기반 리서치 자동화를 돕는 오픈소스 프로젝트입니다.

## Features

- 뉴스 기사 수집
- 뉴스 중복 제거 및 정규화
- 종목명 및 별칭 기반 관련 종목 매칭
- ETF 후보 연결
- AI 기반 뉴스 요약 및 테마 분류
- FastAPI 백엔드
- Streamlit 대시보드
- pytest 기반 테스트 코드

## Tech Stack

- Python
- FastAPI
- Streamlit
- SQLite
- OpenAI API
- pytest

## Disclaimer

이 프로젝트는 투자 조언을 제공하지 않습니다.
뉴스 기반 리서치 자동화와 개발 학습을 위한 오픈소스 도구입니다.
